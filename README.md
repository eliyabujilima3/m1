# Meshack W. Makala — Portfolio Clouds

A professional personal portfolio website for Meshack W. Makala, built with a modern frontend and a simple Flask backend for contact form submissions and admin management.

## Features

- Dynamic responsive landing page
- About, Skills, Services, Projects, Qualifications, and Contact sections
- Contact form with backend submission
- Admin login and message management backend API
- Clean modern design with mobile-first responsiveness

## Setup

1. Create a Python virtual environment inside `backend/`:

   ```powershell
   cd backend
   python -m venv .venv
   .\.venv\Scripts\Activate.ps1
   pip install -r requirements.txt
   ```

2. Copy `.env.example` to `.env` and update values.
3. Initialize the database:

   ```powershell
   python init_db.py
   ```

4. Run the backend server:

   ```powershell
   python app.py
   ```

5. Open the frontend pages directly in a browser or serve them with a static web server.
